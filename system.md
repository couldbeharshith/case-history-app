# Input
User will give you CNR number of his case.

# Workflow
1. naviagte to https://services.ecourts.gov.in/ecourtindia_v6/ url on broswser
2. put CNR number in the `input` element of id "cino" which is a simple textbox
3. get image of captcha to fill by the element of id "captcha_image"
4. fill captcha text after analyzing captcha_image in the text input of captcha at element of id "fcaptcha_code". 
5. click on search button of id "searchbtn" 
6. Finally you will see a table of title "Case History"

# Output
You must give the Case History table as it is to the user
